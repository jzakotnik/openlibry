## Der typische Tag

So sieht ein typischer Tag in der Schulbibliothek aus:

### Ausleihe (während der Pause)

1. Öffne **Leihe**
2. Kinder kommen mit Bücherwünschen
3. Scanne den Ausweis des Kindes (oder suche den Namen)
4. Scanne das Buch (oder suche den Titel)
5. Klicke auf Ausleihen ▶️
6. Fertig – das nächste Kind wartet schon!

### Rückgabe

1. Kind bringt Buch zurück
2. Scanne den Ausweis (optional)
3. Scanne das Buch
4. Klicke auf Rückgabe ↩️
5. "Danke, bis bald!"

### Überfällige Bücher

1. Auf der Leihe-Seite siehst du überfällige Bücher
2. Rot markiert = schon länger überfällig
3. Gelb = bald fällig
4. Bei Bedarf: Mahnschreiben erstellen unter **Reports**

## Tastaturkürzel

Für schnelles Arbeiten:

| Taste | Funktion |
|-------|----------|
| ++esc++ | Suchfeld leeren |
| ++esc++ ++esc++ | Zurück zur Nutzersuche |
| ++enter++ | Suche bestätigen (mit Barcode-Scanner automatisch) |
| ++tab++ | Zum nächsten Feld springen |

## Tipps für Anfänger

1. **Erst Nutzer, dann Buch**: Bei der Ausleihe immer erst den Nutzer auswählen
2. **Barcode-Scanner nutzen**: Spart enorm viel Zeit
3. **Tastatur-Workflow lernen**: Mit etwas Übung geht alles viel schneller
4. **Fehler passieren**: Du kannst alles rückgängig machen
