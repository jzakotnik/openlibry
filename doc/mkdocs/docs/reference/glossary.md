# Glossar

Begriffserklärungen für OpenLibry.

| Begriff | Erklärung |
|---------|-----------|
| **Mediennummer** | Eindeutige ID eines Buchs, wird als Barcode gedruckt |
| **Ausweisnummer** | Eindeutige ID eines Nutzers |
| **Schlagwort** | Kategorie/Tag zur Klassifizierung von Büchern |
| **Leihfrist** | Zeitraum, für den ein Buch ausgeliehen werden kann |
| **Verlängerung** | Erneute Ausleihe ohne Rückgabe |
| **Mahnung** | Erinnerung bei überfälligen Büchern |
| **Antolin** | Leseförderungsprogramm des Westermann Verlags |
| **ISBN** | Internationale Standard-Buchnummer |
| **DNB** | Deutsche Nationalbibliothek |
| **Barcode** | Strichcode zur schnellen Erfassung |
| **SQLite** | Einfache Datenbankdatei ohne Server |
| **Prisma** | ORM (Object-Relational Mapping) für die Datenbank |
