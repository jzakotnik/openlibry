# Changelog

Versionshistorie von OpenLibry.

## Aktuelle Version

Siehe [GitHub Releases](https://github.com/jzakotnik/openlibry/releases) für die vollständige Versionshistorie.

## Versionsschema

OpenLibry verwendet semantische Versionierung:

- **Major** (1.0.0 → 2.0.0): Breaking Changes
- **Minor** (1.0.0 → 1.1.0): Neue Features
- **Patch** (1.0.0 → 1.0.1): Bugfixes

## Highlights

### Neueste Verbesserungen

- Glassmorphism UI-Design
- Performance-Optimierungen
- Verbesserte Barcode-Unterstützung
- Cypress-Tests
- Docker Multi-Arch Support (x86_64 + ARM64)

## Upgrade-Hinweise

Vor jedem Upgrade:

1. Backup erstellen!
2. Changelog lesen
3. Breaking Changes beachten
