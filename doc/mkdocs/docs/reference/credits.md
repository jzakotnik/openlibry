## Open Source Komponenten

OpenLibry verwendet:

- [Next.js](https://nextjs.org/) – React-Framework
- [Material-UI](https://mui.com/) – UI-Komponenten
- [Prisma](https://prisma.io/) – Datenbank-ORM
- [React-PDF](https://react-pdf.org/) – PDF-Generierung
